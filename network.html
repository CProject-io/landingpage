The Free Access Network (Layer 0)

C-Project's long-term vision is the creation of a physical, decentralized, and independent communication infrastructure.

Objective and Philosophy

The goal is not to replace the internet, but to offer a resilient parallel network for essential needs. We believe that access to communication and financial services is a basic right. This network is designed to be censorship-resistant and resilient to outages, ensuring that communities can stay connected and operational even if traditional infrastructures fail.

Technical Operation

The C-Project Network is built as a **mesh network** composed of its autonomous nodes. We use several protocols to achieve this:

  • Local Communication (Mesh): Nearby nodes communicate directly with each other using their Wi-Fi interfaces, forming a local mesh. To manage routing in this mesh, we explore protocols like CJDNS, BATMAN-adv, or Babel.
  • Long-Range Communication (LoRaWAN): To connect nodes or groups of nodes that are geographically distant, we plan to integrate LoRaWAN modules. This allows small data packets to be sent over long distances with minimal power consumption, ideal for transactions or short messages.
  • Interoperability (Layer 0): As a pure network layer, it is designed to be agnostic to the type of data it transports. This means it can serve as the infrastructure for C-Project's own blockchain, but also for nodes of other blockchains (like Bitcoin or Ethereum) that desire a more resilient and decentralized communication layer.

Auxiliary Network Services

For proper functioning, the network will integrate key services such as a **Decentralized DNS** for node discovery without relying on central name servers, and the use of **IPFS** for distributed storage of heavier data.
